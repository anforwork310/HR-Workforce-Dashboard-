# HR-Workforce-Dashboard-
An interactive Power BI dashboard providing insights into employee demographics, turnover, satisfaction, and recruitment performance to support strategic HR decisions.

## I. Introduction
### 1. Introduction to Dataset

- The dataset contains detailed HR employee information including demographics, compensation, tenure, satisfaction scores, performance ratings, and recruitment sources.
- It allows analysis across multiple dimensions such as department, gender, and age group to monitor workforce composition, engagement, and retention.
